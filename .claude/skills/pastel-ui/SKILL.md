# Pastel UI Skill

## 概要

占いアプリのデザインを統一するためのスキルです。すべてのデザイン修正は **パステルカラー基調で角丸の可愛いスタイル** に統一します。

## カラーパレット

### メインカラー
- **ソフトピンク**: `#FFB3D9` - アクセント、ボタン（愛情・優しさを表現）
- **ソフトパープル**: `#D8BFD8` - 見出し、ボックス背景（神秘性・落ち着き）
- **ソフトミント**: `#B3E5D8` - セカンダリーボタン、ホバー効果
- **ソフトクリーム**: `#FFF9E6` - メイン背景、カードの背景
- **ソフトラベンダー**: `#E6D7FF` - グラデーション用の明るい紫

### アクセントカラー
- **ゴールド**: `#FFD700` - 星、重要な要素
- **ペールコーラル**: `#FFB8A3` - 警告・注意の要素
- **グレイッシュブルー**: `#A0B4C7` - サブテキスト

## スタイルガイドライン

### 1. 角丸（Border Radius）
全てのコンポーネントで最低限以下の角丸を適用：

- **コンテナ・カード**: `border-radius: 24px` または `28px`
- **ボタン**: `border-radius: 16px` または `20px`
- **入力欄・表示エリア**: `border-radius: 12px` または `16px`
- **小さいコンポーネント**: `border-radius: 8px` または `12px`

### 2. 背景とグラデーション

#### ページ背景
```
linear-gradient(135deg, #FFF9E6 0%, #F0E6FF 50%, #FFE6F0 100%)
```
やさしい、ふわふわしたパステルグラデーション

#### カード・ボックス背景
- **白基調**: `rgba(255, 255, 255, 0.9)` または `#FFFBF0`
- **グラデーション**: `linear-gradient(135deg, #FFE6F0 0%, #E6D7FF 100%)`

### 3. ボタンスタイル
- **メインボタン（占うボタン）**: ソフトピンク + グラデーション
  ```
  background: linear-gradient(135deg, #FFB3D9 0%, #D8BFD8 100%)
  ```
- **カテゴリボタン（非選択時）**: ソフトクリーム背景
  ```
  background: #FFF9E6
  color: #7D5A8A
  ```
- **カテゴリボタン（選択時）**: ソフトパープル + 白文字
  ```
  background: #D8BFD8
  color: #FFFFFF
  ```

### 4. テキスト
- **見出し（H1）**: `#9B6B9E` (ソフトパープルの濃い色)
- **本文**: `#5A4A5A` (濃いグレー系)
- **サブテキスト**: `#A0B4C7` (グレイッシュブルー)
- **ホワイト（濃い背景上）**: `#FFFFFF`

### 5. シャドウ・エフェクト
- **ソフトシャドウ**: `box-shadow: 0 8px 24px rgba(0, 0, 0, 0.08)`
- **ホバーシャドウ**: `box-shadow: 0 12px 32px rgba(255, 179, 217, 0.2)`

### 6. トランジション・アニメーション
- **ホバー時の遷移**: `transition: all 0.3s ease`
- **表示アニメーション**: `@keyframes fadeIn` で 0.5s ease-in-out
- **ボタンプレス効果**: `transform: translateY(-3px)` on hover

## 使用例

### ボタンの実装
```html
<button class="btn-main" style="
  background: linear-gradient(135deg, #FFB3D9 0%, #D8BFD8 100%);
  border: none;
  border-radius: 20px;
  color: white;
  padding: 15px 30px;
  font-weight: bold;
  cursor: pointer;
  box-shadow: 0 8px 24px rgba(0, 0, 0, 0.08);
  transition: all 0.3s ease;
">
  占う
</button>
```

### コンテナの実装
```html
<div class="container" style="
  background: linear-gradient(135deg, #FFE6F0 0%, #E6D7FF 100%);
  border-radius: 28px;
  box-shadow: 0 8px 24px rgba(0, 0, 0, 0.08);
  padding: 40px;
">
  <!-- コンテンツ -->
</div>
```

- [ ] シャドウはソフトで自然か
- [ ] ホバー・アクティブ状態が可愛らしいか

- パステルカラーの特性：柔らかく、落ち着いた印象を与える
- 角丸デザイン：現代的で親しみやすい雰囲気を作る
- 占いアプリとの親和性：神秘性と可愛らしさのバランス
